# StudyMate AI

AI study companion for university students. Upload PDFs, get summaries, quizzes, flashcards, and study plans.

## Run locally

### Backend
```
cd backend
npm install
npm run dev
```

### Frontend
Open:
```
http://localhost:8080/
```

## Key Features
- JWT authentication
- PDF upload + parsing
- AI summary + quiz + flashcards
- Study planner
- History, dashboard, search
- Dark mode toggle
